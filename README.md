# ITDGram

> Telegram-клиент для ИТД  
> Лента, уведомления, лайки, комментарии и посты — прямо внутри Telegram.

---

## Возможности

- 🏠 Просмотр ленты ИТД
- ❤️ Лайки
- 💬 Комментарии
- 🔔 Красивые уведомления
- 👤 Профиль пользователя
- 🔎 Поиск пользователей
- 📝 Публикация постов
- ✔ Система верификации ITDGram
- 🛠 Админ-панель
- 📢 Рассылка сообщений
- 🔄 Live-уведомления

---

### Как получить refresh_token
1.Открыть итд.com
2.Нажать F12
3.Application
4.Cookies
5.Найти refresh_token

## Верификация ✔
✔ — галочка внутри ITDGram.
Выдается администрацией проекта.
Внутри самого ИТД галочка не отображается.
